Система USB Sensor Monitor состоит из двух компонентов:
	1. Драйвер устройства.
	2. Служба,обеспечивающая интерфейс к драйверу, и позволяющая
		получать состояние сенсорных контактов локально (на той же машине)
		и удаленно. В обоих случаях клиенты подключаются к службе по протоколу TCP.

1. Установка драйвера
	1. Вставьте устройсто в USB порт.
	2. Установите драйвер, следуя указаниям мастера установки устройств,
		в качестве inf-файла укажите usbjk.inf, находящийся в папке Driver.
2. Проверка драйвера
	1. Запустите DriverTest.exe, находящийся в папке Test.
	2. Щелкните кнопку "Open Device".
	3. Если устройство установлено правильно, появится мигающая 
		надпись синего цвета - "Device is connected". Если драйвер
		не установлен, установлен неправильно или устройство 
		отсутствует, появится надпись красного 
		цвета - "Error opening device".
	4. Щелкните чекбокс "Enable polling".
	5. Замыкая и размыкая сенсорные контакты, проверьте работоспособность
		устройства.
3. Установка службы
	1. Служба DevMon.exe (находится в папке Service) устанавливается на
		той же машине, к которой подключается устройство.
	2. Для работы службы требуется наличие .NET Framework 2.0.
	3. Для установки службы запустите файл install.cmd.
	4. Для запуска службы запустите start.cmd.
4. Проверка службы
	Служба ждет подключения на TCP порт 23000. После подключения требуется 
	ввести пароль. После ввода пароля служба начинает передавать строку
	состояния устройства. В исходном состоянии, если не замкнут ни один
	контакт, передается строка "0000". При замыкании какого-либо контакта
	в соответствующей позиции передается не "0", а "1". Так, если замкнуты 
	первый и третий контакты, передается "1010" и т.п.
	
	1. Для проверки работоспособности службы на локальной машине запустите
		командный файл ServiceTest.cmd в папке Test
	2. Наберите пароль.
	3. Наблюдайте ответ службы. Если ни один контакт не замкнут, служба 
		ответит строкой "0000".
	4. Для проверки работоспособности службы по сети отредактируйте 
		командный файл, изменив имя хоста или наберите следующую 
		команду в командной строке Windows: telnet HostName 23000.
